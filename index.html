CBMC

Getting started with CBMC

CBMC is a model checker for C. This means that CBMC will explore all possible paths through your code on all possible inputs, and will check that all assertions in your code are true. CBMC can also check for the possibility of memory safety errors (like buffer overflow) and for instances of undefined behavior (like signed integer overflow). CBMC is a bounded model checker, however, which means that using CBMC may require restricting this set of all possible inputs to inputs of some bounded size. CBMC has been used on over a dozen software projects as part of software development and continuous integration.

This book is a collection of tutorials and reference guides for using CBMC in software verification.
